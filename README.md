# Personal Portfolio

A responsive personal portfolio website with sections for about, skills, projects, blog posts, resume, and contact.

## Features
- Single-page layout with anchor navigation
- Hero, about, skills, projects, blog, resume, and contact sections
- Responsive styling for mobile, tablet, and desktop
- Project and blog data rendered from JavaScript
- Contact form UI with client-side validation hooks

## Project Structure
```
.
├── index.html
├── blog.html
├── assets/
│   ├── styles.css
│   ├── script.js
│   ├── blog-post.js
│   ├── images/
│   ├── css/
│   └── js/
└── pages/
```

## Getting Started
1. Open `index.html` in your browser.
2. Use the navigation links to jump between sections.
3. Blog post pages open from `blog.html` with query parameters.

## Customization
- Update content in `index.html` and `blog.html`.
- Adjust styles in `assets/styles.css`.
- Edit dynamic data and behavior in `assets/script.js` and `assets/blog-post.js`.
- Replace images in `assets/images/` and update the references as needed.

## Deployment
You can deploy this site with any static hosting provider (GitHub Pages, Netlify, Vercel). Point the host to the root folder.
